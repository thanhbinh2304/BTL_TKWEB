HTML & CSS Cơ Bản

Mô Tả Khóa Học:

Khóa học này sẽ giúp bạn làm quen với cách xây dựng giao diện website bằng HTML và CSS. Từ những kiến thức cơ bản nhất đến việc thiết kế các website đơn giản, chuyên nghiệp.

Nội Dung Chính:

  • Giới thiệu về HTML và cấu trúc trang web
  • Thẻ HTML cơ bản và nâng cao
  • Giới thiệu về CSS và cách sử dụng CSS
  • Thiết kế giao diện web responsive
  • Dự án thực hành cuối khóa
